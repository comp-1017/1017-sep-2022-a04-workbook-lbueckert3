CSS Coding Styles

Browser Defaults you have not supplied any css code
Inline this is for a single specific element (tag) instance. The element contains a parameter called style="" style="rule; [rule; ...]" style="color:red; width:50px;"
Embedded Coded in your head area of the webpage. Is started with the style element and contains the selectors and rules for the desired css. Applies only to the physical page where the code is placed. Affects all selectors on the page. <style>
selector {rule; [rule; ...]
[selector {rule; [rule; ...]} ...]
</style>		 <style>
p {font-size:large; font-weight:bold;}
blockquote {width:350px;}
</style>
Stylesheet similiar to embedded, however the selectors and rules are coded in a separate physcial file. This file can be shared thorughout the entire website. Thus, all css is in one common place. selector {rule; [rule; ...]
[selector {rule; [rule; ...]} ...]		 p {font-size:large; font-weight:bold;}
blockquote {width:350px;}
